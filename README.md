# IMDb-Movie-Review-Sentiment-Analysis
Machine Learning project that classifies IMDb movie reviews as positive or negative using Natural Language Processing (NLP) and sentiment analysis techniques.
Overview

This project focuses on Sentiment Analysis of IMDb movie reviews using Natural Language Processing (NLP) and Machine Learning. The model analyzes textual movie reviews and predicts whether the sentiment expressed is Positive or Negative.

The project demonstrates the complete machine learning pipeline, including text preprocessing, feature extraction, model training, evaluation, and prediction.

⸻

🚀 Features

* Text preprocessing and cleaning
* Natural Language Processing (NLP)
* Feature extraction using TF-IDF/Bag of Words
* Binary sentiment classification
* Model evaluation and performance comparison
* Prediction on custom movie reviews

* 😊 Positive Review
* 😞 Negative Review

📊 Project Workflow

1. Load IMDb dataset
2. Clean and preprocess text
3. Tokenization
4. Remove stop words
5. Stemming/Lemmatization
6. Convert text into numerical features (TF-IDF or Bag of Words)
7. Train Machine Learning model
8. Evaluate performance
9. Predict sentiment for new reviews
